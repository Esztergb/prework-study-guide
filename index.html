HTML

  • HTML is the "Frame of the house"
  • HTML stands for Hypertext Markup Language
  • It is made up of elements to provide comtent and framework for a webpage
  • Most common elements are: Text elements, Image Elements, Header Elements and List Elements
  • The head element contains information about the webpage
  • the bodu element represents the visible content shown to the user

CSS

  • CSS gives "color and style to out house"
  • CSS stands for Cascading Style Sheets
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: check what branch we are currently on
  • git checkout -b branch-name: vreates a new branch and switches to it
  • git clone (paste copied ssh values): copy content
  • git add -A: add modification to the current working branch (-A flag indicates to add all changes)
  • git commit: take changes that made locally and puch them up to a remote branch
  • git pull origin main: pull down any changes from the base branch
  • git push origin feature/starter-code: push our change to the remote branch

JavaScript

  • JavaScript gives "the windows and doors" to our house to make it interactive
  • JavaScript adds logic and user interactivity to websites
  • Each line in JavaScript ends with semicolon.
  • All strings are surrounded by quotation marks
